# Acumen Discord Ranking Bot

> This Discord Bot will specialize to give exp and rank up your users and reward them with crypto!

## Exp Calculation
> lvlExpRequirement + (lvlExpPercentage * lvlExpRequirement)

It is simple calculation.Though I am not thinking of putting some limitation on lvl that user can get.
The variable will be this:
- lvlExpRequirement is the lvl that you need to lvl up
- lvlExpPercentage is the multiplier

Check this [pastebin](https://pastebin.com/Y7jxHpDy) for example output of calculation.

## Goal

- Ranking System
- Guild Seperation for Users that are in multiple guild that the bot is currently on.It also will reward you Mintme for everytime you lvl up!
